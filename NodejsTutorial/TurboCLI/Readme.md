# REST API With Node JS and Mongo DB (Udemy)

## Setup turbo cli
[Turbo 360 CLI](https://www.turbo360.co/feed)
```
npm i -g turbo-cli
turbo version
```
## Create project
```
turbo new rest-api
```

## Run project
```
turbo devserver
turbo app [Site ID]
turbo login
turbo deploy
```
